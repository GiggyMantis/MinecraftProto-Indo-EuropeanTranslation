# Proto-Indo-European translation of the hit video game Minecraft (2009)

## Style Guide
- Follows [Wiktionary's transcription system](https://en.wiktionary.org/wiki/Wiktionary:Proto-Indo-European_entry_guidelines)
  - Mobile s is thus written as (s)
- Include the preceding asterisk at the start of reconstructed terms
- Minecraft-copyright terms should not be changed (Netherite, Creeper, etc.)
- If only dialectical terms exist for a word, choose one of the dialects and hope it evens out in the end.
- If a root is spurious or controversial, try to avoid it
